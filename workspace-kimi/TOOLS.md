# Kimi Analyst — Tools

## Available Tools
- Full filesystem access (read-focused)
- Shell execution for file listing, diffing, grepping
- memory_search — search workspace memory
- memory_get — read specific memory files

## Useful Commands
- `find <dir> -name "*.md" | head -50` — discover documentation
- `wc -l <file>` — gauge file sizes before reading
- `git log --oneline -20` — recent changes
- `git diff <ref1> <ref2> --stat` — change summary between refs
- `tree -L 2 <dir>` — directory structure overview
