# DeepSeek Challenger — Tools

## Available Tools
- Full filesystem access (read-focused)
- Shell execution for investigation (git log, grep, test runners)
- memory_search — search workspace memory
- memory_get — read specific memory files

## Useful Commands
- `git log --oneline --all -30` — recent history across branches
- `git blame <file>` — who changed what and when
- `grep -rn "<pattern>" <dir>` — search for patterns
- `git diff <ref>` — examine specific changes
